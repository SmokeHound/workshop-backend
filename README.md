\# 🛠 Workshop Backend



Express + SQLite backend API for managing workshop consumables and orders. Built for frontend integration via Netlify.



\## ✨ Features

\- `POST /api/login`

\- `POST /api/save-order`

\- `GET /api/items`

\- `GET /api/history`

\- `GET /api/stats`



\## ⚙️ Setup



```bash

npm install

node index.js
